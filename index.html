Welcome to Vulcan

Introduction

Vulcan is a advanced control for CNC Laser cutting machines which is optimized for

  • Easy intuitive shop floor operation

  • Programming and program modification on the machine

  • Advanced technology function to work robust with high laser powers

  • Modular: Grow from a standalone machine into a fully automated machine

Home screen

When you start, the above page is displayed. From this page you are guided with the workflow:

* Setup Jogging and other setup steps (→ Setup)

* Create Load DXF parts or create nests (→ Create)

* Programs Select a existing part or nest (→ Programs)

* Controls Select machine functions (→ Controls)

* Run Run a program (→ Run)

* Configure Change machine settings (→ Configure)
