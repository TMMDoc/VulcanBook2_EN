Program selector

Selecting the program and pressing the RUN Key navigates the user to the run tab. The program name, dimension, NCR status, and laser parameters (axes details, speed, gas pressure, power, gas type, nozzle height, focal distance, current work coordinate system, tool coordinate system, M function used) are shown at the top.

  • Start : Starts the program.

  • Pause : Pause the program. This option is only available when the user starts the program.

  • Continue : Continues the paused program.

  • Stop or Abort : Stops the program.

  • Edit : Edits the program. It navigates to the program’s edit page which was mentioned earlier.

Speed override

The user can use the Override + and Override - to influence the speed of axes movements (in percent).

  • Override + : Increase override percentage for the configured/set axis or path speed. Above 10%, the value is changed in 10% increments; below 10%, it is changed in 1% increments.

  • Override – : Reduce override percentage for the configured/set axis or path speed. Above 10%, the value is changed in 10% decrements; below 10%, it is changed in 1% decrements. The user can also use the slider to increase or decrease the speed.

User can navigate between different tabs while the machine is running:

It is possible for the user to move out of “Run” page even while the machine is busy cutting the sheet. This is necessary because, while a program is running, the operator might want to get ready with the next set of programs (edit program, change laser parameters etc) for production. Additionally, when the user switch tabs while program is running, a mini player is displayed for him/her to know the current state of program being cut, without moving to “Run” tab

Display

User can turn ON or OFF the sheet, machine and traverse display in the drawing.

PLC static messages are displayed at the bottom left corner in RUN page

Status

User can check the information of program and machine status. It’s a read only type.

Program controls

User can control the program by choosing the required option when program is in RUN state. Options chosen in program controls will be displayed in bottom left corner.
