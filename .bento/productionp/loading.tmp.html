Sheet Edge Measure

The Sheet Edge Measure function should be adjusted to adapt both Single job mode and Production plan mode.

Single Job mode: Keep current workflow. Positioning CU → Start Production → Acknowledge the Popups → Start Sheet Measuring

Production plan mode: There should not pop any acknowledgement message for further user actions, that means we should automatically execute the Sheet Edge Measuring function while production starts. No CU positioning necessary → Start Sheet Measuring based on Sheet Stop Corner and combined with the measuring distance offset. (Distance can be adjusted in MK, to adapt certain special cases.)

Loading Type

Automation Configuration While there’s automation device connected, the production plan will be influenced by automation interface signals. (e.g, Loading finished, Unloading finished…) So, we have to provide a configuration to activate/deactivate the automation configuration. The parameter should able to be changed by customer, so it could be put in the General page of the Machine configurations.

Loading Type: a. Manual Loading without Pallet exchange - This is to work with Single program job b. Manual Loading with Pallet exchange - This is to work with Production plan c. Loading with External loader - This is to work with External loader

Sheet Stop Position

Sheet loading position definition The sheet stop for raw material stack in entire machines’ layout could be set in different corners. Automation device has the accordant sheet stop corner, thus the sheet loading position in machine will be impacted as well.

In this case, we should have the parameter to set the loading corner so that Control can calculate the Sheet Edge Measuring accordingly. Run a single program can still keep the current implementation, Sheet Edge Measuring for Single program running will not be impacted.

There will be 4 selections in the drop-down selection. - Position 1 - Position 2 - Position 3 - Position 4

Once it be configured, in production plan mode, the Sheet Edge Measuring strategy will be adjusted accordingly.
