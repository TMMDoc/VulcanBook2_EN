Add a production plan

This function is to add production plan packages into the production plan list. - Single file, Multiple files should be support while importing. - Source file should support from Local disk, USB, Network disk. - Can refer to the current solution for importing programs. - The split production plan packages will be into several single programs and stored as well in original Program page. So the operators can use them to do single job processing.
