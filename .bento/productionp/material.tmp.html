This function is to provide the Material status on machine pallets based on the production plan process. Operator should observe the actual status and compare to adjust here if necessary.

Location: Inside and Outside pallets are displayed.

Name: The sheet information will be read once the production plan starts, not able to edit.

Action: The needed action will be displayed and can be adjusted by drop-down selection. - Will be processed - Empty - Will be unloaded

Will be processed: This is to indicate the sheet will be set as raw material/re-entry to be processed.

Empty: No material on the machine pallets.

Will be unloaded: This is to indicate the sheet will be set as finished sheet/scrap sheet to be unloaded.

The current Material Stock status will be real-time read from Control based on the running production plan. Control will read the information while production plan start, and the following actions will be executed by pallet/automation.

This is a very important feature while Automation connected. Normally if the production plan be aborted, continue production could face different scenarios. Sometimes, the job be produced temporally in single job mode, then the Material Stock adjustment is very useful for operators. While adjustment be finished, Control will process the further actions and the sheets could be moved by Automation as expected.
