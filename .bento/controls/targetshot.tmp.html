By activating this program, it helps to shoot the beam with peak power in maximum beam mode. The procedure to check the program is to have a small piece of the black chart with some distance below from the nozzle. By calling the program, the beam generates with the specified parameter at the defined position. Once this operation is done check the spot image from the black chart.

With the above entries, X and Y axis moves towards the positioned value.

Running the program will follow the data of power, frequency and duty as per the user requirement.

Focus diameter, Focus distance and dwell time also has been modified according to the user requirements. Processed values can be monitored when the program runs. Dwell time specified will hold the program until the dwell time configured and then program aborts.
