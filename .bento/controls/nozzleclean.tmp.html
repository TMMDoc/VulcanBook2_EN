By activating this program, it cleans the nozzle at the nozzle brush point and calibrate the height sensor on an external plate located near the brush point.

Axis will move to the defined offset values configured in machine settings.

Navigate to Configure → Machine settings → PLC options → Control parameters → Nozzle cleaning & height sensor calibration offsets.

To activate auto nozzle clean options, navigate to Configure → Machine settings → PLC options → Function parameters → Auto Nozzle clean options.

It performs as per the configuration of auto nozzle clean options.
