By activating this program, it helps to check the focal distance manually. It asks to enter focus diameter and focus distance. User can also monitor the active focus diameter and distance.

Entered amount of focus distance will be displayed as below:

If Focus diameter is selected as “150um”, Focus shift value is predefined as “2.5mm”, Focus distance will revert back to “2.5mm” when program gets aborted.

If Focus diameter is selected as “210um”, Focus shift value is predefined as “0mm”, Focus distance will revert back to “0mm” when program gets aborted.
