By activating this program, user can check the variation of focus position by choosing the required technology table according to the usage of sheet with captured work offset. Minimum and Maximum focus, Serial number, Focus diameter and shift value can be defined according to the user requirement.

Minimum and Maximum focus of diameter 150um and 210um is -3 to +3 respectively.

Focus Shift1 value is to determine the focus position. User can adjust the value according to the progress by checking it in feeler gauge.

Determine the smallest kerf “K” (= focus position) in the focus comb.

  • With the suitable feeler gauge (e.g., 0.25 mm), starting from the left and right, respectively, check every individual kerf.

  • Mark the kerf where the feeler gauge sticks on both left and right sides.

Calculate the focus shift and enter it under “Focus Shift1”.

Example

Old focus shift A mm, smallest kerf K mm.

Focus shift = A + K.

User can also monitor the focus diameter during program running time.

In order to calibrate the focus position exactly towards the sheet surface, the focus comb is used. The focus comb is a programm which varies the focus height and adjusts the offset caused by the tolerance of the optical components.
