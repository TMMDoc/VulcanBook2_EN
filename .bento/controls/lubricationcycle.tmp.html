By activating this program, it helps to run lubrication for a defined period.

The Lubrication cycle program will support continuous lubrication cycles.

Lubrication cycles defines the repeat times of lubrication, default setting = 1.

Lubrication interval defines the intervals between each lubrication, default setting = 1s.

The change will not impact the auto triggering Lubrication configured in machine settings.

Navigate to Configure → Machine settings → PLC options → Control parameters → Auto lubrication cycle.
