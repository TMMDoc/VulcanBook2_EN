Edge Detection function is to calculate the origin point and rotation angle of the sheet.

According to the rotation of the sheet, it is required to rotate the XY-plane of the program such that the jobs are fitted into the rotated sheet.

Syntax: G17 D=Sheet Rotation

To enable this function, navigate to Configure → Machine settings → PLC options → Function Parameters.

Enable Activate sheet edge function from Machine Miscellaneous Control.

It is also possible to manually program the Edge Detection function.

Switch to Controls page to select the Edge detection.

By selecting Edge detection program, Operator must enter the sheet length of X and Y axis to detect the edges.

For the current automatic edge detection function (three-point edge-seeking), operator needs to accurately enter the size of the sheet, and the cutting head needs to be precisely located near the corner of the sheet at the same time. Or else the edge detection may fail.

After the running the edge detection function, the program will go to the origin point and move the Z-axis down to the corner and wait for the acknowledgement of the operator to the start the program.

To enable this function, navigate to Configure → Machine settings → PLC options → Control Parameters.

Select the Acknowledge the sheet origin point checkbox.

With this option selected (same as current behavior):

  1. Head move to origin point

  2. Move the Z-axis to down position

  3. Wait for operator to acknowledge by cycle start

Without this option selected:

  1. Pilot diode will be moved to origin point

  2. Program will be Dwell for 3 secs

  3. Program start to continue.
