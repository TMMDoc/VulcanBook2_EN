HMI Simulation

In this page we run the simulation for the program. Program name and its dimension are displayed at the top. We can play, pause, and reset the simulation. The slider shows the progress of simulation. It can be dragged to the left to rewind or right to forward the simulation.

This button is used to zoom in the program.

This button is used to zoom out the program.

This button is used to move the program to the top.

This button is used to move the to the bottom.

Selecting done will close the simulation window.
