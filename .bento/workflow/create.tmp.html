This page displays all the available DXF files.

Create Job

The user needs to select any of the DXF files or multiple files and then select the create job button. After a Job is created, you will be presented with a tabbed interface as shown in the screenshot below. There are four tabs in this window as follow:

  1. Parts tab

  2. Configuration tab

  3. Sheets tab

  4. Layouts tab

Parts tab

In this, the user can add as many parts as they want. They need to select all the dxf files and select create job to add all the parts. In the parts tab the user can change max quantity, set priority, grain direction, part rotation and the group. In this tab you will manage the list of the parts (and their required quantities) that must be nested.

Configuration tab

Allows you to modify the parameters such as material, thickness, film, treatment and part gap.

Sheets tab

This is the tab where you will manage the list of sheets to be considered for nesting. The user can add sheets with different sizes. Increase or decrease the quantity. The margin can also be changed.

Use Twinline if you need efficient processing of your parts

Layouts tab

When you select the Layout tab, Vulcan will automatically start the nesting computation, if there are no layouts. Once the computation is done, it will present the computed layouts as a list of thumbnails. You can edit any of these layouts in the Flux main window by clicking on the thumbnail. You get a full-fledged layout editor where you can modify the position/orientation of parts, edit tooling, edit sequence etc.

Hit the Recompute button to command Vulcan to nest remaining (un-nested parts). If you wish to re-nest from start, delete existing layouts and then hit this button.

Selecting the edit button will open the edit page for layout which is explained in detail in section 3.3 and Selecting the cancel button to not save the changes done here.

The user can save the layouts using the save layouts. Selecting it opens a window for the layout name. Enter the name and click ok to save the layouts. The save layout will be available in programs page.
