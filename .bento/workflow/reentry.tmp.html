In case you had to stop a program in the previous run Vulcan allows you to re-entry this sheet again to save the material.

  • Continue from stop point should resume the program

  • Restart from Beginning should begin a new run

  • Cancel should go back to the previous screen (no running the program)

Re-entry point allows the user to restart an interrupted program from a specific point. User can select the particular line in the DIN editor or directly select the segment in the drawing or use the “goto” option. Block input has the option of entering the line and block number manually in block input or by choosing the segment, it will update. Within the selected segment, the slider can be used to position the point, exactly from which the program has to be executed.

In the “goto” option the user can navigate between different contours or segments. After selecting the point and clicking done, activates the block search mode. When you press START, the machine moves to the position from which the program restarts. After approaching the start position, the following message appears: “Block search target position reached. Press START to continue”. Now press START to begin processing.
