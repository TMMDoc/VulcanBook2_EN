The captured point will be considered as the zero point of the active coordinate system. You can either enter the value manually or choose the mode respectively to configure as work offset.

A table of all coordinate systems is displayed (except S0). For each coordinate system, the offset is indicated relative to S0. These offsets can be changed within the table by either entering the values manually or choose the mode respectively to configure as work offset. Clicking on done, saves the changes in the controller.
