Definition

Programming The TC_FLUID_OFF cycle deactivates water spraying. TC_FLUID_OFF | Water spraying is generally controlled via the machine technology tables (program properties). Mode 0 or 1 switches water spraying on or off generally in the overall program. If in the machine technology table mode 2 is active, water spraying can be activated in a program-controlled manner via the cycle TC_FLUID_ON and deactivated via TC_FLUID_OFF. Deactivate TC_FLUID_ON. Is deactivated by program end or program cancelation. Contour related: must be programmed after TC_LASER_OFF. =
