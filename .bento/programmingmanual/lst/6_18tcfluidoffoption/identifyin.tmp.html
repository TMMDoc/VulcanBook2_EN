Definition

Programming Variable text can be engraved on the workpiece with the TC_WRITE cycle. The TC_WRITE cycle outputs the text at the current position in positive X direction. The laser technology table, the font height in mm and the text direction in degrees can be changed via the cycle’s corresponding transfer parameter.

Notes ■ Plane processing should be selected for round tubes. ■ The function is available for 2D laser machines with the "Siemens Solution-Line 840D" control. TC_WRITE(T_TEXT,T_TABNAME,T_HEIGHT,T_ANGLE) ■ TC_WRITE must be programmed before contour machining (TC_LASER_ON). |

|Parameter a|

Value

Definition |T_TEXT a| Text to be engraved on the workpiece (max. 255 characters). |T_TABNAME a| Name of the laser technology table. |T_HEIGHT a| Font size in mm or inch. |T_ANGLE a| Default value: 0° Optional parameter for text rotation in degrees. = Transfer parameters Tab. 5-16
