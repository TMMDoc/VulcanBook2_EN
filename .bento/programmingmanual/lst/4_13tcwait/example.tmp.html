1.5 TC_POS_MODE

Definition

Programming The TC_POS_MODE cycle controls the positioning process of the cutting head between the cutting processes. TC_POS_MODE (Value) = ■ Self-holding (modal) operation. ■ Is deactivated by the renewed programming of the cycle, the end of the program or the actuation of the Reset button.

=

Value

Explanation Standard behavior. Unsupported retraction of the contour after TC_LASER_OFF. Prerequisite for the programming of this value is that programming be done without piercing and with TC_LASER_OFF(4). ■ Floating approach of the contour with TC_LASER_ON. ■ Unsupported retraction of the contour after TC_LASER_OFF. = Tab. 5-12
