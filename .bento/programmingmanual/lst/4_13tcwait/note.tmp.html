1.4 G823 (FlyLine)

Definition

Programming Programming a contour-related beam suppression. With the G823 function, the laser beam can be suppressed in relation to the block. At the same time, the beam at the block transition is switched off unsupported. If another travel block without a G823 function follows a travel block with this function, the beam will be ignited at the block transition unsupported. The respective process parameters remained unchanged thereby. At the same time, the beam suppression is not restricted to linear blocks.

Note = The G823 function is assigned to a group of G commands. Only one G command may be used from this group within an NC block. The G commands G820 – G829 belong to this group. ■ The function only becomes effective if the laser processing was previously activated by TC_LASER_ON. ■ A piercing process must be possible without axis standstill when the material is selected or the material thickness is selected. ■ Depending on the machine, FlyLine can be used for various material thicknesses (for exact information on the material thickness, see the data collection for the machine concerned).
