Microjoints

Definition Microjoints are microwebs that could usually be programmed only with the programming system TruTops. The microjoints defined in this way are permanently anchored in the NC program. They cannot be modified or deselected during the machining of parts at the machine. Depending on the machine, it may also be possible to program microjoints directly on the machine and select or deselect them as required. There are two different types of microjoints:

■ Programmed microjoints that are defined within the contour or at the end of the contour via G functions. The contour is marked here directly in the NC text for the part program.

■ Automatic microjoints that are always set at the end of the contour and defined without G functions. The definition is made here via the dimension of the contour in X- and Ydirection. The contour is not marked in the NC text. The following points are valid: ■ Microjoints that can only be modified at the machine are only possible on the machine on surfaces of rectangular tubes or for plane processing. ■ The data for programmed microjoints is stored in a programspecific table. The data can be displayed and modified via the user interface of the control. This table can be used to activate and deactivate the microjoint types separately. The table is called up using the NC cycle TC_MICROJOINT ("table name"). ■ The activation status of the microjoints in the table is stored with the program. In this way, an available program is guaranteed to be processed under the same conditions after every program change, even when fully automatic systems are used. Call up table with microjoint data directly at the control system and activate or deactivate programmed- or automatic microjoints in the NC text. = The cycle TC_MICROJOINT in the NC program calls up the table containing all the parameters and the activation status of the functions. Data can be modified in the table. The modified table is updated automatically in the NC core.

Programming TC_MICROJOINT("Table name"): ■ Self-holding (modal) operation. ■ Call up as early as possible before the required contour machining. ■ Is deselected using program end or with the selection of a new table.
