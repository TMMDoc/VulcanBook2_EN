With the TC_TIMER cycle, machining times for single workpieces can be measured, displayed, and logged. A maximum of five timings can be simultaneously programmed, and displayed together later. TC_TIMER (INT P1, INT NUMBER, STRING[20] COMMENT) ■ The laser beam must not be ignited. ■ A maximum of five timings can be started parallel to each other, or one after the other. The sequence is arbitrary. ■ It takes 0.5 seconds to start each timing. If several timings are started simultaneously, then these 0.5 seconds are added to each timing. ■ The maximum measurable time is nine hours.
