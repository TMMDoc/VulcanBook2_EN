Every start of the contour will contain the block number for easy identification of the contour, tracking of error in the program etc., For Block Search function, the block number need to be written to P-field (BlockNo, internally defined the P-field) for easy identification of blocks to transfer the parameters to the PLC.

Syntax: N1

BlockNo=1

  • The Block number should be specified for a Closed contour only. If any closed contour is split into more than one, then all should lie in same block number.

  • Refer the above chapter for details of the block number.
