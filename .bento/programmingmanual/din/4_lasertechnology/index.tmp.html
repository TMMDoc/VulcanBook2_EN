Depends on the technology, the DIN program structure can be modified to suit the requirements of the user. Using the Laser Core 800 CNC controller, a DIN program for the Laser technology is composed of following:

  • Program Header

  • Technology Table Transfer

  • Check for Boundary program active

  • Block Jump

  • Block Number

  • Start of a Part Information

  • Laser parameters selection & Update

  • Contour statements

  • Program End

Example Program:

%1 (PP_Sample_Pgm.DIN)_ CNC_ID=1 Sheet_Size_X = 120 Sheet_Size_Y = 600 Job_Size_X = 105 Job_Size_Y = 573 X_Limit_Minus = 10 X_Limit_Plus = 115 Y_Limit_Minus = 10 Y_Limit_Plus = 583 G17 D=SheetRotation Program Header G253 F="TT:MS_1mm" E0
