For a more automated workflow you could use the production plan. The production plan is a list of programs that you want to run.

A production plan can consist of none, one or more than one production package arranged in a sequence. A production package can contain several jobs. (One LST includes several programs) A job contains an NC program with any number of repetitions. The target quantity defined in the programming system will be set

At automated machines, the production plan controls the continuous material flow processing, from raw material loading, processing, and finished parts unloading.
