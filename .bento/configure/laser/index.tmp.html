To show LTT data in Vulcan, change the “Machine Database Type” from Defaults to Trumpf LTT. Then Vulcan read LDM-Cloud_TC49.XML which is generated from Trumpf’s Laser Database Management (LDM) and show the LTT data. User can edit the required laser parameter values for each page.

Selecting the field opens a numerical keyboard. Using this we can enter the values for the parameters.

  • Clone page : User can clone a current laser page parameters and create a new laser page with different material, thickness, lens and nozzle type.

  • Delete page : User can delete a laser page which is not required.

  • Import : User can import a new laser data LDM-Cloud_TC49.XML by selecting the respective directory.

  • Export : User can export a current laser data LDM-Cloud_TC49.XML by choosing the respective directory to save it.

  • Update : User can select this button to update LTT. This updated new LTT will replace “TrumpfDefaults.XML” file and Add new LTT table and data to LDM-Cloud_TC49.XML and Defaults.XML files.

  • View defaults : User can select this button to View default values of the parameter which stored in Defaults.XML files, if the parameter is different from default values.

  • Save and Restore defaults : User can save the values as defaults either for "current" or "all LTT’s" and restore later from "user" or "Trumpf defaults".

  • Done : Saves all the changes.

  • Ramp cycles : Vulcan read LDM-Cloud_TC49.XML which contains ramp cycles information and show the ramp cycle data as follows.

With reference to Trumpf LTT, the Ramp cycle parameters are used for the Piercing. A maximum of 199 ramp cycles are available in which from 181 to 199, customer can use it.
