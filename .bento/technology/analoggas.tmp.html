If you want to use a proportional valve with analog DC 0-10v control for air pressure, you need to use air pressure calibration where the proportional valve air pressure feedback is a DC Voltage.

Steps to follow:

  1. Gas Configuration: Select the gas config of the machine as follows:

  1. Select the calibration for the analogue control of pressure valve:

  1. Check the values of High Pressure & Low pressure valves.

  1. Select the Gas Purge program, if the calibration is activated in the PLC options, then the Calibration option will be displayed in this program:

  1. When the calibration is selected, the table will be displayed as follows:

Important points of the Calibration screen:

  1. The calibration table will be stored for each gas type.

  2. Max pressure value was displayed depends on the Gas configuration and Pressure values entered in "Custom Parameters 2".

  3. Calib Mode: Manual → Press start button for next output, Auto → For defined delay will output the next value

  4. If Feedback is selected, then the Valve feedback is saved for the monitoring of the gas pressure, else will save the set value no monitoring)

  5. DA Auto fill → will fill the DA output depends on the number of calibration points.

  6. When the calibration is started, enter the gas pressure in the pop-up window for each output value.

  7. Can start the calibration from the selected index also.
