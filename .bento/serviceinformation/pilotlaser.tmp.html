An external pilot laser is equipped onto the cutting unit to indicate the starting position of the selected cutting program. The beam path of this pilot laser is in parallel with laser beam used for cutting. And due to the mechanical construction, the pilot laser beam in X/Y plane has a certain offset in comparison with the beam used for cutting. This offset in X/Y plane varies from machine to machine, due to the mechanical structure’s accuracy issue and assembly. Therefore to automatically correct this offset when operator used pilot laser to define the starting position, this offset needs to be checked in both X and Y direction and saved in machine’s parameter during machine assembly process, with the help of a Pilot Laser Calibration Program and it is implemented in Vulcan as:
