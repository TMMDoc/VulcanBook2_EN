5. Kerf correction

=

Definition

Definition

Programming

Contour change

Correction direction Kerf correction enables the machining of a part-related program on an equidistant to the programmed path. A path with a constant distance to the programmed contour is approached.

5.1 TC_LASERCORR_ON Activating the kerf correction. The TC_LASERCORR_ON cycle call-up is called up by using a parameter in the NC program: TC_LASERCORR_ON (Correction direction) ■ Is programmed after TC_LASER_ON is called up. ■ The following NC block must be a linear block (G01). The correction must be retracted on a linear movement. ■ The tool path of the laser is calculated from the "kerf" value from the technology table. The kerf is derived from the cutting parameters of TC_LASER_ON. ■ Is deleted by TC_LASER_OFF (Beam OFF, automatic switch off of the kerf correction and establishment of the last programmed position of the laser cutting head). Kerf correction does not have to be switched off if a contour change with a modified kerf is to be performed on a correction which has already been extended. A TC_LASER_ON call with corresponding cutting type need only be programmed. The new correction is corrected linearly in the subsequent motion block for the entire block. The kerf correction is carried out to the left or right of the programmed contour, depending on the set parameters. = Kerf correction
