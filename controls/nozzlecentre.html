Nozzle Centre

Nozzle Centre

By activating this program, it helps to check the Centre point of the nozzle where the beam generates. The procedure to check the program is to apply some grease to the nozzle tip and cover the nozzle with white tape. By calling the program, the beam generates with the specified parameter at the defined position. Then, remove the white tape and see the beam spot is in the middle of the nozzle tip ring.

With the above entries, X and Y axis moves towards the positioned value.

Running the program will follow the data of power, frequency and duty as per the user requirement.

Focus diameter, Focus distance and dwell time also has been modified according to the user requirements. Entered values can be monitored when the program runs. Dwell time specified will hold the program until the dwell time configured and then program aborts.
